# Book Store Application

## Overview

Book Store Application is a web-based system developed to manage books, customers, and book orders efficiently.

## Features

- Book Management
- Customer Management
- Order Processing
- Search Books
- Database Integration

## Technologies Used

- JavaScript
- Node.js
- Express.js
- MongoDB

## Project Structure

- index.js - Main server file
- package.json - Project dependencies
- package-lock.json - Dependency lock file

## Database

MongoDB Atlas is used as the database solution.
